📱 手机DNS配置指南

让你的手机使用 http://labubu.local:3000 访问壁纸画廊

🎯 配置目标

将手机DNS设置为: 172.27.2.191

配置成功后，手机可以使用友好的域名访问Labubu壁纸画廊

🤖
Android 手机

📱 通用步骤 (适用大部分Android手机)

  1. 打开手机 设置
  2. 点击 WiFi 或 无线和网络
  3. 找到当前连接的WiFi网络
  4. 长按WiFi名称，选择 修改网络
  5. 点击 高级选项 或 显示高级选项
  6. 将 IP设置 改为 静态
  7. 在 DNS 1 中输入: 172.27.2.191
  8. 点击 保存
  9. 断开WiFi重新连接
🍎
iPhone 手机

📱 iOS系统配置步骤

  1. 打开 设置
  2. 点击 WiFi
  3. 点击当前WiFi网络旁的 ℹ️ 图标
  4. 点击 配置DNS
  5. 选择 手动
  6. 点击 添加服务器
  7. 输入: 172.27.2.191
  8. 点击 存储
  9. 关闭WiFi再重新连接

📋 不同品牌手机的具体路径

🔸 华为/荣耀
设置 → WLAN → 长按WiFi → 修改网络 → 显示高级选项
🔸 小米/红米
设置 → WLAN → 点击WiFi → IP设置 → 静态
🔸 OPPO/一加

🔸 vivo
设置 → WLAN → 点击WiFi → IP设置 → 静态

设置 → 连接 → WiFi → 点击WiFi → 高级 → IP设置
🔸 魅族
设置 → WLAN → 点击WiFi → 静态IP → DNS设置

🧪 配置完成后测试

在手机浏览器中访问以下地址，如果能打开说明配置成功：

🐰 http://labubu.local:3000 🎨 备用域名1 🖼️ 备用域名2

🔧 故障排除

❌ 找不到DNS设置选项

  • 不同手机品牌界面略有差异，寻找关键词：DNS、域名服务器、网络设置
  • 有些手机在【连接】→【更多连接设置】中
  • 部分老旧手机可能不支持修改DNS

❌ 配置后仍无法访问

  • 检查输入的IP是否正确：172.27.2.191
  • 尝试重启手机WiFi或重启手机
  • 确认手机和电脑在同一WiFi网络
  • 尝试使用IP地址访问：http://172.27.2.191:3000

💡 替代方案

🌐 路由器DNS配置 (推荐)

在电脑浏览器访问 http://172.27.0.1 登录路由器管理界面

添加静态DNS条目：labubu.local → 172.27.2.191

✅ 优势：一次配置，所有设备自动生效

📱 DNS修改应用

Android可以使用：DNS Changer、1.1.1.1等应用

⚠️ 部分应用需要root权限

🔢 继续使用IP访问

如果DNS配置困难，直接使用IP访问

地址：http://172.27.2.191:3000

✅ 功能完全相同，只是地址不够友好
